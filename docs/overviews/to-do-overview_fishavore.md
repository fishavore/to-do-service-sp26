# To-Do Service API

The To-Do Service is an http-based task management service.
It helps users add tasks and receive reminders of them.

The API simulates the REST interface of the To-Do Service.

## Get started

Ready to try the To-Do Service API? Jump onto this `Quickstart` (coming soon).

Wanna learn more? [Tutorials](/docs/tutorials.md) is the right place.
Instructions to set up your system are at [Before you start a tutorial](/docs/before-you-start-a-tutorial.md).

## API reference

Already know what you want? You can find descriptions of the resources are in
the API reference.

In the docs, `{base_url}` refers to the URL of a resource. The default is
`http://localhost:3000`.

* [user resource](/docs/api/user.md)
* [task resource](/docs/api/task.md)
