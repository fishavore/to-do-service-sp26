# Advantages and limitations of Markdown

The [Daring Fireball](https://daringfireball.net/projects/markdown/) website says:
>Markdown is a text-to-HTML conversion tool for web writers.

It's easy to create a simple HTML page using Markdown, but many HTML features are difficult to
apply with Markdown only.

## Recommendation for using Markdown

According to Module 3 Lesson 2, there are dialects or variations of Markdown. Depending on the tool
to write and publish the content, it's a good idea to find a formatting guideline of Markdown for
that specific tool[^1].

[^1]: This content's Markdown follows [Markdown Cheatsheet](https://github.com/adam-p/markdown-here/wiki/Markdown-Cheatsheet).
